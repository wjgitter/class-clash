# class-clash
A two player fighting game! Knock your opponent off the edge!
|                | Player 1 | Player 2          |
|----------------|----------|-------------------|
| Move           | A or D   | Left or Right Key |
| Jump           | W        | Up Key            |
| Normal Attack  | F        | ,                 |
| Special Attack | G        | .                 |

Made with Canva AI. No mobile support yet. Sorry not sorry.
